# Customer-churn-analysis
An interactive dashboard that analyzes customer behavior and identifies key factors driving churn, helping businesses improve retention and make data-driven decisions.
📊 Customer-Churn-Analysis (Interactive Dashboard)
🎯 Project Objective

The objective of this project is to analyze customer churn behavior and identify the key factors responsible for customer attrition. This helps businesses improve customer retention, reduce churn rate, and enhance overall customer satisfaction.


❓ Questions (KPIs)
What is the overall churn rate?
Which customer segments have the highest churn?
Does contract type affect churn rate?
What is the impact of monthly charges on churn?
Which services are most associated with churn?
Churn distribution based on tenure
Gender-wise churn analysis
Payment method vs churn relationship
⚙️ Process
Verified dataset for missing values and handled inconsistencies
Cleaned and transformed data (data types, formatting, null values)
Encoded categorical variables where required
Created calculated measures (Churn Rate, Customer Segments, etc.)
Built pivot tables / visualizations based on KPIs
Designed an interactive dashboard using slicers/filters (Contract Type, Gender, Tenure, Payment Method)
Combined all visuals into a dynamic dashboard

📈 Project Insight
Customers with month-to-month contracts have a higher churn rate
Higher monthly charges are associated with increased churn
Customers with shorter tenure are more likely to churn
Electronic check payment method shows higher churn compared to others
Certain services (like lack of tech support or online security) increase churn probability
Long-term contract customers are more stable and less likely to churn
✅ Final Conclusion

To reduce customer churn, businesses should focus on converting short-term customers into long-term contracts by offering incentives and loyalty programs.

Additionally, improving service quality, promoting bundled services, and encouraging automatic payment methods can significantly reduce churn and improve customer retention.
